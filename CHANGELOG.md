# Changelog

All notable changes to Cloud Auditor will be documented in this file.

## [2.0.0] - 2026-03-11

### Added
- Initial public release of the hardened CLI.
- RSA-signed security scan responses.
- Support for Multi-Cloud compliance mappings (SOC 2, HIPAA, CIS).
- Premium HTML report generation.
- Hardware-bound licensing system for enterprise security.

### Changed
- Improved scanning performance for large AWS environments.
- Enhanced reporting engine with dynamic visualizations.

### Fixed
- Resolved minor edge cases in S3 bucket policy analysis.
