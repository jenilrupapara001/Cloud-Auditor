**High-Intensity. 100% Local. Uncrackable.**

Cloud Auditor is the industry-leading AWS security auditing engine designed for elite security teams who demand absolute privacy and military-grade hardening.

## 🚀 Key Features

*   **100% Local Audit**: Your AWS credentials never leave your terminal. No cloud-to-cloud connections.
*   **RSA-Signed Security**: Every scan response is cryptographically signed for integrity and authenticity.
*   **Hardware Binding**: Advanced licensing bound to unique hardware IDs to prevent unauthorized use.
*   **Instant Compliance**: Automatic mapping to SOC 2, HIPAA, CIS, and ISO 27001.
*   **Beautiful Reports**: Generate premium HTML reports with dynamic visualizations and actionable insights.

## 📥 Get Started

Download the latest hardened binary for your platform from the [Releases](https://github.com/USER/cloud-auditor/releases) page.

### Quick Setup

```bash
# 1. Download the binary (e.g., cloud-auditor-darwin-arm64)
# 2. Make it executable
chmod +x cloud-auditor-darwin-arm64

# 3. Run a scan
./cloud-auditor-darwin-arm64 audit aws --region us-east-1
```

For more detailed instructions, see [INSTALL.md](INSTALL.md).
